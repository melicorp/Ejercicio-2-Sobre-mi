Se explicaran los siguientes términos: función, let, const y tres etiquetas de html que no hayamos visto en sesión.

  • Función: En programación, una función es un bloque de código que realiza una tarea específica y puede ser llamado o invocado desde otras partes del programa. Las funciones se utilizan para dividir un programa en partes más pequeñas y manejables.
  • let: En JavaScript, "let" es una palabra clave utilizada para declarar variables con alcance de bloque. Esto significa que la variable solo está disponible dentro del bloque en el que se declara, lo que ayuda a evitar conflictos de nombres.
  • const: En JavaScript, "const" es una palabra clave que se usa para declarar constantes. Las constantes son variables cuyo valor no puede cambiar una vez que se les asigna un valor inicial. Son útiles para definir valores que no deben modificarse en el programa.
